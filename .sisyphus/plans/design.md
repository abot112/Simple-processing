# ImageTool GUI - 设计文档

## 项目概述

一个基于 PyQt6 的图像处理GUI工具，提供图像相减、局部放大和图像排版三个核心功能。

---

## 技术栈

- **语言**: Python 3.10+
- **GUI框架**: PyQt6
- **图像处理**: Pillow (PIL), NumPy, OpenCV
- **项目结构**: 模块化设计，易于扩展

---

## 设计决策汇总

| 问题 | 决策 |
|------|------|
| **启动方式** | start.bat 直接启动 |
| **支持格式** | PNG, JPG/JPEG, BMP |
| **相减算法** | 绝对值相减 \|像素1 - 像素2\| |
| **区域选择** | 矩形框选（鼠标拖拽） |
| **放大样式** | 带引线的放大镜效果（学术论文风格） |
| **拼图尺寸** | 预设格式 + 自定义行列数 |
| **拖拽交互** | 拖拽 + 点击选择（两者支持） |
| **保存选项** | 每次处理选择保存文件夹 |
| **配色方案** | 标准Claude暖色风格 |
| **窗口布局** | 多标签页（顶部Tab切换） |

## 组件规范

### 按钮样式

**主按钮（Primary）**
- 背景: `#D97757`（暖橙）

- 边框: 无
- 圆角: 6px
- 悬停: `#E8C4B8`（浅暖橙）
- 内边距: 10px 24px

**次级按钮（Secondary）**
- 背景: `#FFFFFF`（白色）
- 文字: `#2D2A26`（深炭）
- 边框: 1px solid `#E8E0D5`
- 圆角: 6px
- 悬停: `#FAF6F1`（米白）

### 输入框样式

- 背景: `#FFFFFF`（白色）
- 边框: 1px solid `#E8E0D5`
- 圆角: 4px
- 聚焦: 边框变为 `#D97757`
- 内边距: 8px 12px

### 标签页样式

- 未选中: 背景 `#FAF6F1`，文字 `#6B6560`
- 选中: 背景 `#FFFFFF`，文字 `#2D2A26`，底部边框 `#D97757` 2px
- 悬停: 背景 `#F0EBE3`

### 图像预览区

- 背景: `#F5F3EF`（浅灰米）
- 边框: 1px dashed `#D0C8BD`（虚线边框）
- 悬停（有文件）: 边框变为 `#D97757`
- 提示文字: `#9B9590`（浅灰）
